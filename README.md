五子棋 (15*15)

简介:
- 这是一个在终端运行的五子棋（Gomoku）实现，使用 Python 3 编写。
- 支持本地双人(人 vs 人) 和 人机(人 vs 电脑) 对战。
- 棋盘为 15x15，行列编号为三位数字，例如 001 到 015。
- 最新落子会在上方的列编号上方显示 ▼，在左侧对应行的左边显示 ▶。
- 使用栈实现悔棋：双人对战每次悔棋回退一步；人机对战悔棋回退两步（悔棋一次等同于撤销你和电脑最近的两步）。
- 每次下棋时会打印“执X棋...”，并在落子后打印“刚刚执X棋 在 RRR CCC 下子”。悔棋时打印“已悔棋”。

运行方法:
1. 确保已安装 Python 3
2. 在终端运行

操作说明:
- 输入: row col （例如: 10 11） 表示在第 10 行第 11 列下子（行列均从 1 开始）
- 输入 u: 悔棋（人机模式下会悔两步）
- 输入 q: 退出游戏

注意:
- AI 为简单启发式实现，并非强力引擎，仅用于练习与娱乐。
