# 五子棋 (Gobong) 15x15

终端五子棋，Python 3 实现。支持本地双人对战和人机对战。

## 运行

```bash
python3 Gobang.py
```

需要 Python 3.6+，无第三方依赖。

## 游戏模式

- **双人对战 (PvP)** — 本地两人轮流下棋
- **人机对战 (PvE)** — 选择先手/后手 + AI 难度

## AI 难度

| 难度 | 策略 |
|------|------|
| 简单 | 基础模式评分 + 随机扰动，优先堵棋，不会纯随机落子 |
| 中等 | 模式评分 + 双重威胁检测（活四/活三组合） |
| 困难 | 全面模式评分 + 双重威胁 + 对手最佳应对模拟 |

评分体系识别棋型：连五 > 活四 > 冲四 > 活三 > 冲三 > 活二 > 冲二

## 操作

| 输入 | 说明 |
|------|------|
| `row col` | 下子，例如 `10 11`（行列从 1 开始） |
| `u` | 悔棋（人机模式回退两步） |
| `q` | 退出 |

## 棋盘

- 15x15，行列编号 001~015
- `▼` 标记最新落子列，`▶` 标记最新落子行
- 棋盘下满时判平局

## AI 实现

- 候选位置裁剪：只考虑已有棋子周围 2 格（24~50 个 vs 全盘 225 个）
- 困难模式：基础评分筛选 top 15 候选 + 深度分析（对手模拟）
- 必赢/必堵优先检测
